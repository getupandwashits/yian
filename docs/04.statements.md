# 04 Statements

此文档介绍 YIAN 中函数体以及方法体中可以使用的语句类型

## Variable Declaration

YIAN 中的变量声明的格式为:

```text
<type> <name> = <expression>
```

- `<type>`: 变量的类型, 可以是任何有效的类型
- `<name>`: 变量的名称, 必须是一个有效的标识符
- `<expression>`: 变量的初始值, 必须是一个类型为 `<type>` 的表达式, 关于表达式的详细信息请参见[表达式](0X.expressions.md)章节
- 初始值是可选的

下面的代码片段展示了变量声明的示例:

```text
i32 a = 10
f64 b = 3.14
bool flag = true
char ch = 'A'
str name  // 未初始化的变量
```
