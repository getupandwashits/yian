# 02 Type System

YIAN 的类型系统包括以下几个大类

1. **基本类型**: 构成类型的基础, 包括算术类型等
2. **派生类型**: 由其他类型派生而来, 包括数组, 指针等
3. **自定义类型**: 用户定义的类型, 包括 struct, enum, trait, 函数类型等

## Basic Types

YIAN 支持以下基本类型:

- `i8`, `i16`, `i32`, `i64`: 有符号整数类型, 分别占用 8, 16, 32, 64 位
- `u8`, `u16`, `u32`, `u64`: 无符号整数类型, 分别占用 8, 16, 32, 64 位
